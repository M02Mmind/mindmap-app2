# 🧠 Mindmap Lernplattform

## Starten (lokal)

1. Node.js installieren (falls noch nicht): https://nodejs.org (LTS-Version)

2. Im Terminal in diesen Ordner wechseln:
   cd mindmap-app2

3. Abhängigkeiten installieren (einmalig):
   npm install

4. App starten:
   npm run dev

5. Im Browser öffnen: http://localhost:5173
